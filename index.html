SECRET C0DE GAME

Your Mission: Crack the Secret Code.

Someone has sent a secret code that needs to be cracked. Maybe it's the code to enter a secret room full of secret international government secrets. Maybe it's something super dangerous, like an explosive that needs disarmed. Maybe your Postmates order is here and the guy needs to door code to get buzzed in. I don't know.

The point is: it's up to you to crack the secret code by guessing five unique digits each turn. For each digit you guess in its correct place, you will receive a "*". For each digit you guess that is in the secret code but is in the wrong place, you will receive a "!". For each digit you guess that is nowhere in the secret code, you will receive a "0".

When you're ready, head to the main menu using the button below.
